Features

Database
Provides a PostgreSQL database with the ability to subscribe to updates through the JavaScript SDK.
Authentication
Supports various sign up and registration flows and can provide row level security.
Storage
Store and serve files. Supports any media, including video files and images.
Functions (soon)
Create and schedule custom functions to run, allowing for serverless architecture.

Database: They handle scaling for you, should you need that

Authentication types: Google, GitHub, GitLab, Azure, Facebook, Twitter, Bitbucket
